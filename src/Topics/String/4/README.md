# Длина последнего слова

Дана строка `s` содержащая слова и пробелы, верните длину последнего слова в строке.
Слово - это подстрока, состаящая из символов, не являющихся пробелами.

Пример 1
```shell
Input: s = "Hello World"
Output: 5
```

Пример 2
```shell
Input: s = "   fly me   to   the moon  "
Output: 4
```
