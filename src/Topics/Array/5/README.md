# Диапазоны чисел

На взод подается отсортированный список из уникальных элементов.

Верните диапазоны числел в массиве

Пример:
```shell
Input: nums = [0,1,2,4,5,7]
Output: ["0->2","4->5","7"]
```


```shell
Input: nums = [0,2,3,4,6,8,9]
Output: ["0","2->4","6","8->9"]

```
