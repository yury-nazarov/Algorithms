# Наибольшая возрастающая последовательность

На вход подается не отсортированный массив `[int]`. Верните длину наибольшей **непрерывно возрастающей последовательности**



Пример 1
```shell
Input: nums = [1,3,5,4,7]
Output: 3
```

Пример 2
```shell
Input: nums = [2,2,2,2,2]
Output: 1
```
