# Суммирование последовательности чисел

Дан массив `nums[int]`. определите последовательность сумм чисел в нем.

Пример 1
```shell
Input: nums = [1, 2, 3, 4]
Output: [1, 3, 6, 10] 
```

Пример 2
```shell
Input: nums = [1, 1, 1, 1, 1]
Output: [1, 2, 3, 4, 5]
```

Пример 3
```shell
Input: nums = [3, 1, 2, 10, 1]
Output: [3, 4, 6, 16, 17]
```
