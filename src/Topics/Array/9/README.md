# Монотонный массив

Массив называется монотонным, если он последовательно возрастает или последовательно убывает

Пример 1
```shell
Input: nums = [1, 2, 2, 3]
Output: true
```

Пример 2
```shell
Input: nums = [6, 5, 4, 4]
Output: true
```

Пример 3
```shell
Input: nums = [1, 3, 2]
Output: false
```
